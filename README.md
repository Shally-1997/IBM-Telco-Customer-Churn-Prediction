IBM Telco Customer Churn Prediction
Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project aims to predict whether a telecom customer is likely to leave the company using machine learning techniques.

The goal is to help businesses identify high-risk customers and take proactive actions to improve customer retention.

Problem Statement

Telecom companies lose customers due to various factors such as high charges, poor service experience, and contract preferences. Predicting customer churn can help reduce customer loss and increase revenue.

Dataset

Dataset: IBM Telco Customer Churn Dataset

The dataset contains customer demographic information, account details, subscribed services, billing information, and churn status.

Target Variable:

Churn (Yes/No)
